__Shell Basics__
* 0-current working directory file is a script that prints the absolute path name of the current workinb directory.
* 1-listit file is a script that displays list of contents in the current directory. 
* 2-bring me home file contains a script that changes the working directory to the user's home directory. 
* 3-listfiles displays contents in the current directory in a long format. 
* 4-listmorefiles displays the current directory's contents including hidden files (starting with .) in a long format. 
* 5-listfilesdigitonly displays current directory contents in a long format, with user and group IDs displayed numerically and hidden files (starting with .)
* 6-firstdirectory contains a script that creates a directory mamed "my first directory" in the /tmp/ directory. 
* 7-movethatfile is a file that has the file, "betty", being moved from /tmp/ to /tmp/my_first_directory. 
* 8-firstdelete consisted of a file named "betty" that was deleted from the directory stated above. 
* 9-firstdirdeletion is the file that had the "my first directory" directory being deleted from the /tmp/ directory. 
* 10-back consists of a script that changes the workinh directory to the previous one. 
* 11-lists consists of a script that lists all files (including those that are hidden and begin with period character) in the current directory and the parent directory and fhe /boot direcrory in long format (in this order).
* 12-file_type consists of a script that prints a filed named "iamafile" which will then be in the /tmp directory when this script is run. 
* 13-symbolic_link is a file which has a symbolic link to /bin/ls and named __ls__. The symlink is jn the current working directory. 
* 14-copy_html consists of a script rhat copies all HTML files from the current working directory to the parent working directory, but only copies files that did not exist in the parent working directory. 
* 100-lets_move contains a script that moved all files beginning with an uppercase letter to the directory /tmp/u. 
* 101-clean_emacs consists of a script that deletes all files in the current working directory that ends with the character "~". 
* 102-tree contains a script that creates these directories; welcome/, welcome/to/ and welcome/to/school in the current directory. 
* 103-commas consists of a command that lists all the files and directories of the current directory, separated by ",". The commands are as follows: *Directory names end with a "/" , *Files and directories starting with dot are listed , *Listing are alpha ordered except directories . and .. that are lusted at the very beginning , the listing ends with a new line. 
* school.mgc consists of a magic file that was created with the name "school.mgc". It can be used wifh the command "file" to detect School data files. The School data files always contain the string SCHOOL at offset 0.
